mojo-executor by TimMoore

The Mojo Executor provides a way to to execute other Mojos (plugins) within a Maven plugin, allowing you to easily create Maven plugins that are composed of other plugins.

This is how you would execute the "copy-dependencies" goal of the Maven 2 Dependency Plugin programmatically: {% highlight java %} executeMojo( plugin( groupId("org.apache.maven.plugins"), artifactId("maven-dependency-plugin"), version("2.0") ), goal("copy-dependencies"), configuration( element(name("outputDirectory"), "${project.build.directory}/foo") ), executionEnvironment( project, session, pluginManager ) ); {% endhighlight %} The project, session, and pluginManager variables should be injected via the normal Mojo injection: {% highlight java %} /** * The Maven Project Object * * @parameter expression="${project}" * @required * @readonly */ protected MavenProject project; /** * The Maven Session Object * * @parameter expression="${session}" * @required * @readonly */ protected MavenSession session; /** * The Maven PluginManager Object * * @component * @required */ protected PluginManager pluginManager; {% endhighlight %}

Install

Add this to your pom.xml: {% highlight xml %} mojo-executor-repository Mojo Executor Repository for Maven http://twdata-m2-repository.googlecode.com/svn/ {% endhighlight %}
